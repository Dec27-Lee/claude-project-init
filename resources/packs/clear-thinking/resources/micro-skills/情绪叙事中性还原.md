# 情绪叙事中性还原

> 动作类型：事实校准
> 场景标签：个人、生活、关系、复盘

## 适用处境

用户把现实纠葛讲成强烈戏剧：必须有情绪高潮、必须有人给说法、必须证明某种意义；但现实可能只是拖延、冷淡、疲惫、惯性、琐碎约束和有限行动。需要把情绪叙事还原为中性事实，再判断解决、等待、沟通、切断或和解。

## 不适用处境

存在明确暴力、霸凌、违法、权益侵害、操控、医疗或安全风险；用户正在寻求情绪支持而非判断；事实尚未查清；当前需要的是立即行动或专业介入。

## 输入

- 用户对事件的情绪化讲述、关键事实、参与方和时间线。
- 用户期待得到的说法、道歉、解释、改变或结果。
- 已尝试的沟通、行动、等待和边界。

## 操作步骤

1. 写出戏剧情绪版本：这件事在用户心里被讲成了什么故事。
2. 改写为中性事实版本：只保留谁、何时、做了什么、产生什么后果。
3. 列出琐碎约束：时间、钱、流程、疲惫、信息不对称、能力不足、惯性和现实限制。
4. 标出真实伤害与可行动部分：哪些需要表达、补证、沟通、修复、切断或求助。
5. 标出不可解决但可承认的部分：无法追回应答、无法补齐意义、无法让对方成为理想角色。
6. 若继续解释不改变下一步，转 `解释关闭`；若边界反复失效，转 `切断力`。

## 关键自问

- 我是在处理事实，还是在寻找一个更有戏剧张力的解释？
- 如果这件事只是中性的、琐碎的、无奈的，下一步会是什么？
- 哪些情绪背后有真实伤害，不能被“中性化”抹掉？
- 我期待的“说法”会改变行动，还是只会延长纠葛？
- 现在该解决、等待、表达、切断，还是接受它不会按剧情收尾？

## 输出格式

```markdown
## 情绪叙事中性还原
- 戏剧情绪叙事：
- 中性事实版本：
- 琐碎约束：
- 真实伤害：
- 可行动部分：
- 不可解决但可承认部分：
- 建议切换：解决 / 等待 / 沟通 / 切断 / 和解 / 求助
- 失败/切换信号：
```

## 常见误用

- 用“现实中性”压掉真实伤害、合理愤怒或维权需求。
- 把所有情绪表达都降级为戏剧化，导致麻木和自我否定。
- 事实不足时急着劝自己接受。
- 用本技能长期忍受明确不公或关系伤害。

## 失效信号

存在暴力、胁迫、霸凌、违法、权益侵害、医疗或安全风险；事实关键缺口会改变行动；用户需要的是情绪陪伴而非判断；还存在必要告知、道歉、修复、交接或维权。命中时切换到 `约束与风险扫描`、`事实-推断-假设分离`、`痛感分流` 或专业支持。

## 最高风险边界

不能把“中性还原”用成劝忍、冷漠、压抑或受害者责任化；凡是伤害、权益和安全问题，先保护边界和证据。

## 常组合

- `事实-推断-假设分离`：把情绪叙事中的事实和解释拆开。
- `痛感分流`：判断痛感来自身体、关系、结构还是行动受阻。
- `解释关闭`：中性还原后停止无效解释。
- `切断力`：关系或事项持续反复且边界失效时。
- `最小行动`：把可行动部分转为下一步。
