# 朋友请求边界问询

> 动作类型：关系承接
> 场景标签：关系、个人、组织、生活、工作、人情、转介、担保

## 适用处境

朋友、同事、亲戚提出模糊帮忙、转介、照应、挂名、推荐、担保等请求，但责任范围、风险、成本和退出条件不清。

## 不适用处境

低风险、低成本、可立即完成的小忙；公共安全、紧急救助或基本生活保障请求；对方只是表达情绪而非提出可执行请求。

## 输入

- 请求内容、关系类型、对方真实期待。
- 时间、金钱、职业、法律、隐私、声誉、担保风险。
- 自己可承接的动作和不可承接的责任。
- 对方是否能接受边界与拒绝。

## 操作步骤

1. 把“帮个忙”翻译成具体事项：做什么、做到什么程度、持续多久、谁承担结果。
2. 标出不可碰线：法律责任、资金担保、职业背书、隐私转交、名义挂靠、长期无偿承诺。
3. 区分情义、人情、雇佣、责任和担保，避免用关系词遮住真实责任。
4. 让请求方承接动机、成本和后果；自己只承接边界内行动。
5. 给出可接受版本、拒绝版本或转介版本；必要时书面确认。

## 关键自问

- 如果这件事出问题，我是否会被视为责任方？
- 我承诺的是信息、建议、陪同、背书、担保，还是结果？
- 对方是否接受边界，还是只接受无条件帮忙？
- 我是否因为不好意思拒绝，把不该承接的风险接过来了？
- 有没有权力不对等、情感压力或经济依赖？

## 输出格式

```markdown
- 请求翻译：
- 可承接事项：
- 不承接事项：
- 风险提示：
- 推荐回应：
- 是否需要书面确认：
```

## 常见误用

- 把边界问询变成审问对方动机。
- 用“理性边界”掩盖自己不愿直接拒绝。
- 在关系压力明显时仍假设双方能平等协商。
- 把一次性帮忙扩大成长期责任。

## 失效信号

救急、公共安全、低风险小事不必过度问询；问询变成羞辱或审判动机时停止；对方无法理解或无法拒绝边界时，不继续谈条件，改为降低承诺、转介或退出。

## 最高风险边界

不能替他人做实质背书、担保、隐私转交、挂名承诺或职业推荐结果承诺，尤其不能用“朋友关系”绕过责任确认。

## 常组合

- `约束与风险扫描`：请求涉及钱、法律、声誉或不可逆承诺时。
- `资本控制权校准`：涉及合伙、出资、挂名或控制权时。
- `会议签字固化承诺`：多人承诺需要书面化时。
- `最小行动`：只需给出低风险回应话术时。
