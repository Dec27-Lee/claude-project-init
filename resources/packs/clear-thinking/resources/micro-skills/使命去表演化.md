# 使命去表演化

> 动作类型：目标取舍
> 场景标签：经营、增长、创业、长期判断、工作、生活、公益、情怀、品牌叙事

## 适用处境

项目带有公益、情怀、社区、帮扶、理想色彩，但需要在市场中成立，且容易把使命变成对外表演、道德压力或模糊披露。

## 不适用处境

法律、募捐、政策、受益方权益、重大利益冲突要求透明披露；当前问题是合规治理或权责安排，不是叙事校准。

## 输入

- 价值目标、商业目标、产品/服务质量。
- 对外叙事、内部机制、受益方预期。
- 必须披露的信息、利益冲突和合规要求。

## 操作步骤

1. 区分内部使命和对外成交理由：内部价值可以存在，但不能替代产品价值。
2. 检查产品/服务去掉同情、情怀、身份标签后是否仍有独立价值。
3. 把价值目标写入内部机制：分配、质量、采购、用工、服务约束和长期承诺。
4. 标出不可使用的话术：道德压力、感恩叙事、身份标签、模糊受益方、隐瞒利益关系。
5. 明确必须披露的信息，不用“低调”掩盖合规和受益方权益。
6. 若商业成立条件不足，先改产品/服务，不用使命补洞。

## 关键自问

- 用户购买的是产品价值，还是被叙事压迫？
- 价值目标有没有进入机制，而不只是进入文案？
- 去掉情怀标签后，产品/服务是否仍值得买？
- 哪些信息必须透明说明？
- 我是在减少表演，还是在隐藏应该披露的事实？

## 输出格式

```markdown
- 内部使命：
- 对外商业成立条件：
- 产品独立价值：
- 内部约束机制：
- 禁用叙事：
- 必须披露信息：
```

## 常见误用

- 以“去表演化”为名隐藏应披露信息。
- 把价值目标完全排除在经营之外。
- 用道德感替代质量、价格和交付。
- 用受益方身份、苦难或标签压迫交易。

## 失效信号

法律要求公开披露、公益募捐、政策项目、受益方权益说明、重大利益冲突时，不能隐藏真实目标；切换到透明披露、合规治理或用户朋友。

## 最高风险边界

不能以“减少表演”为由隐瞒必须披露的信息，也不能用价值叙事、身份标签或道德压力压迫交易。

## 常组合

- `用户朋友`：回到用户真实价值和交易理由。
- `可信度资产冷启动`：把价值目标转为可查证据。
- `约束与风险扫描`：涉及披露、合规或利益冲突时。
- `方案取舍`：使命和商业目标冲突时。
