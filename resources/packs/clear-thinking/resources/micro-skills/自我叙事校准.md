# 自我叙事校准

> 动作类型：事实校准
> 场景标签：个人、复盘、生活、方法建设

## 适用处境

用户提供自述、年度总结、生日总结、长期日记或阶段复盘，文字中有强烈身份标签、使命感、受害感、英雄感、失败叙事、清醒叙事或“我就是这样的人”的结论；需要从叙事里拆出事实、阶段、价值和行动，而不是直接共情、美化或否定。

## 不适用处境

用户只要润色文风、纪念性表达或纯创作；材料事实不足且无法定位具体事件；存在急性心理危机、人身安全、医疗、法律或家庭暴力等风险；当前已经有明确低风险行动，不需要再拆叙事。

## 输入

- 用户原文或阶段总结中的关键句、关键事件和反复出现的身份标签。
- 叙事服务的目标：理解自己、做选择、复盘方法、写作表达，还是进入行动。
- 已知事实、情绪判断、价值偏好和现实约束。

## 操作步骤

1. 抽出叙事中的 3 类句子：发生了什么、我如何解释它、我因此给自己贴了什么标签。
2. 标注叙事功能：保护自己、获得理解、解释失败、维持理想、关闭痛苦、召唤行动，或只是文学表达。
3. 找出遮蔽行动的标签：凡是让下一步变模糊、让责任无法分配、让阶段变化被忽略的自我定义，都降级为待验证叙事。
4. 保留仍有用的价值核心：把“我是什么样的人”改写为“我在什么条件下想保护什么”。
5. 形成当前版本：事实是什么、阶段是什么、我选择保留什么、停止什么、下一步用什么行动验证。
6. 若叙事拆完仍只剩痛苦或疲惫，转 `痛感分流`；若行动已明确，转 `最小行动` 或 `解释关闭`。

## 关键自问

- 这句话是在描述事实，还是在维护一个我习惯讲述的自己？
- 这个身份标签帮我行动，还是帮我停在原地？
- 哪些痛苦被包装成了使命、深刻、清醒、独立或成熟？
- 如果把“我就是……”改成“在……条件下我会……”，判断是否更准确？
- 这个叙事版本需要保留、升级、停用，还是等新事实出现再复查？

## 输出格式

```markdown
## 自我叙事校准
- 核心叙事：
- 支撑事实：
- 推断/情绪解释：
- 身份标签：
- 叙事功能：保护 / 解释 / 表达 / 行动 / 逃避 / 其他
- 遮蔽的行动或事实：
- 可保留的价值核心：
- 当前版本改写：
- 下一步验证：
- 失败/切换信号：
```

## 常见误用

- 把用户的文学表达全部当成问题来拆，破坏纪念性和审美表达。
- 用“自我叙事”否定真实伤害、真实努力或现实约束。
- 把所有理想、善意、浪漫都解释成表演。
- 只做精致解读，不导出行动、边界或复查条件。

## 失效信号

拆完叙事后不改变行动；用户明确只要写作润色或纪念表达；材料缺少具体事件；出现心理危机、医疗、安全、法律等专业边界；分析开始让用户否定全部自我价值。命中时停止本技能，转 `痛感分流`、`事实-推断-假设分离`、`解释关闭` 或专业支持。

## 最高风险边界

不能把个人叙事分析做成心理诊断、人格评判或创伤解释；不能鼓励用户用“我叙事不准”来否认被伤害、被压迫或需要求助的事实。

## 常组合

- `事实-推断-假设分离`：先区分事实、解释和价值偏好。
- `痛感分流`：叙事背后有强痛感且原因混杂时。
- `方法论版本化复盘`：旧自我方法或原则在新阶段失灵时。
- `阶段轮廓与转折点管理`：需要把叙事放进阶段变化中复查。
- `最小行动`：把校准后的版本落成下一步验证。
