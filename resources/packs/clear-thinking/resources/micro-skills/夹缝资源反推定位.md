# 夹缝资源反推定位

> 动作类型：目标取舍
> 场景标签：经营、增长、创业、长期判断、工作、生活

## 适用处境

资源弱、品牌弱、技术弱、资金少，无法与强者正面竞争，但手里有被主流低估的资源、场景、耐心或低固定成本优势。

## 不适用处境

没有真实需求，只是想避开竞争；资源优势依赖低薪、无偿、情感绑架、关系动员或不可持续透支；只能靠低价、低保障维持。

## 输入

- 自身资源、强者优势和不能硬碰的领域。
- 市场缝隙、用户痛点、产品可分解性。
- 固定成本、抗波动能力、合规边界和失败承受力。

## 操作步骤

1. 列出强者优势和自己不能硬碰的地方，先排除正面战场。
2. 找主流参与者不愿做、不经济、不灵活或不愿承受摩擦的细分场景。
3. 盘点可转化资源：时间、场地、人工、关系、场景、耐心、低固定成本、服务细节。
4. 匹配能放大这些资源的产品或服务假设。
5. 验证需求是否真实、用户是否愿意为稳定质量付费，而不只是想要便宜。
6. 预演淡旺季、扩张瓶颈、质量控制和失败成本。

## 关键自问

- 我真正拥有但别人不愿投入的资源是什么？
- 这个缝隙是机会，还是低质量需求陷阱？
- 如果需求增长，我的结构能否承接？
- 用户愿意为稳定质量付费吗，还是只想占便宜？
- 我是否把他人透支当成自己的资源？

## 输出格式

```markdown
- 不硬碰区域：
- 可进入缝隙：
- 低估资源：
- 产品/服务假设：
- 成本结构：
- 验证方式：
- 放弃条件：
```

## 常见误用

- 把低价内卷误认为定位。
- 把熟人关系、无偿劳动或低保障当作可持续资源。
- 找到小缝隙后过早扩大固定成本。
- 只证明有人想要便宜，没有证明有人愿意为质量付费。

## 失效信号

无真实需求、只是低价内卷、质量不可控、合规门槛高，或依赖低薪、无偿、情感绑架、关系动员、不可持续透支人时不用。

## 最高风险边界

不能把他人的低保障、无偿付出、关系压力或不可持续透支当作自己的竞争资源。

## 常组合

- `顺势判断`：判断外部趋势是否真实。
- `用户朋友`：确认缝隙中真实用户需求。
- `规律代价对齐检查`：核对定位需要承接的代价。
- `最小行动`：设计低成本验证。
