# 痛感分流

> 动作类型：关系承接
> 场景标签：关系、个人、组织、生活、工作

## 适用处境

用户反复表达不舒服、委屈、焦虑、愤怒、累、卡住，但原因混杂；同一件事同时牵涉身体状态、关系互动、工作安排、责任边界或行动阻塞。

## 不适用处境

用户已经明确要具体执行步骤；存在紧急医疗、心理危机、人身安全风险；当前需要先查证合同、日志、数据等硬事实；有人试图用“痛感”替代责任判断。

## 输入

- 触发事件：谁、在什么场景、做了什么、造成什么影响。
- 事实证据、身体状态、关系状态、工作状态。
- 已尝试的沟通、休息、行动或边界设置。

## 操作步骤

1. 记录具体触发点，不先解释人格、动机或长期命运。
2. 分流痛感类型：身体疲惫、关系受损、结构信号、行动受阻、价值冲突。
3. 判断痛感是否可行动化：休息、沟通、调整规则、拆下一步、止损或暂缓决策。
4. 为主要类型选择承接方式：身体先恢复，关系先边界，结构先规则，行动先拆步，价值冲突先取舍。
5. 设置复查点：做一个小动作后观察痛感是否下降、事实是否变清楚、风险是否扩大。

## 关键自问

- 我痛的是事情本身，还是持续被迫用错误方式承接事情？
- 这是一次性不适，还是重复出现的模式？
- 如果我休息充分，这个判断会不会改变？
- 有没有具体事实支撑“这是结构问题”？
- 我是在寻找解决路径，还是寻找继续忍耐的理由？
- 这个痛感是在提醒我行动、沟通、休息，还是退出？

## 输出格式

```markdown
## 痛感分流
- 触发事件：
- 主要痛感：
- 初步归类：身体疲惫 / 关系受损 / 结构信号 / 行动受阻 / 价值冲突
- 事实依据：
- 暂不确定之处：
- 建议承接方式：休息 / 沟通 / 行动 / 调整结构 / 止损
- 下一步最小动作：
- 复查信号：
```

## 常见误用

- 把所有痛苦心理化，忽视制度、资源、权责问题。
- 把所有不适结构化，逃避自身能力、沟通或执行问题。
- 在身体严重疲惫时做长期判断。
- 用“我感觉不好”直接推出“必须退出”。
- 用分析痛感替代行动，越分析越陷入解释循环。

## 失效信号

用户只要具体执行，不需要情绪/结构判断；分流后不改变下一步行动；事实不足但无法补充具体事件；已出现安全、医疗、法律等高风险问题；连续多轮只解释痛感却没有休息、沟通、边界或行动。

## 最高风险边界

不做诊断式心理判断，不替代医疗、心理、法律或安全支持；不鼓励用户在高激动或严重疲惫时做不可逆决定；不要求用户继续承受明显伤害来“验证”。

## 常组合

- `事实-推断-假设分离`：防止感受直接变成结论。
- `身体边界与学徒性`：判断高压投入是否已越界。
- `切断力`：痛感反复且边界失效时进入止损。
- `主要矛盾`：从混杂痛感中抓主要冲突。
- `最小行动`：把分流结果落到可验证动作。
