# 可信度资产冷启动

> 动作类型：增长信任
> 场景标签：经营、增长、创业、长期判断、工作、生活

## 适用处境

新产品、新品牌、新团队、新个人履历缺少信任，但本身确有价值，需要建立可验证、可追问、可持续的外部信任。

## 不适用处境

产品、服务或个人能力尚不能承接基本审查；证据主要依赖虚假包装、不可查背书或夸大承诺；真正缺的是质量改进而不是展示材料。

## 输入

- 目标对象及其认可的证据类型。
- 已有产品事实、案例、测评、认证、作品、试用反馈。
- 可展示渠道、可补证动作和不可使用表述。

## 操作步骤

1. 明确目标对象凭什么相信：案例、试用、第三方评价、作品集、认证、稳定交付记录等。
2. 盘点已有可查事实，区分证明、展示、承诺和包装。
3. 排除不可查背书、夸大证明和无法承接追问的材料。
4. 设计低成本证据链：小范围试用、可复现结果、用户反馈、过程记录、第三方验证。
5. 把证据嵌入说明书、销售页、投标材料、简历或作品集。
6. 若质量撑不住，先转入产品/能力改进，不做信任包装。

## 关键自问

- 目标对象凭什么相信我？
- 这条证据是否可查、可复现、可追问？
- 如果对方深问，我是否能承接？
- 哪些表述是在暗示我没有证明的能力？
- 我是在建设信任，还是制造信任幻觉？

## 输出格式

```markdown
- 目标对象：
- 认可的证据类型：
- 已有可信资产：
- 证据缺口：
- 补证计划：
- 展示位置：
- 禁用表述：
```

## 常见误用

- 把可信度建设做成营销包装。
- 先找背书，后补事实。
- 用少量案例暗示普遍能力。
- 把愿景、态度或关系当作已经发生的证据。

## 失效信号

产品质量撑不住、证据虚假、背书不可查、只包装不改事实时不用；证据链无法经受目标对象追问时，不应扩大传播。

## 最高风险边界

不能用不可查背书、夸大证明、身份包装或暗示性表述制造虚假信任。

## 常组合

- `用户朋友`：确认目标对象真实信什么。
- `事实-推断-假设分离`：区分证据和包装。
- `口碑飞轮`：把可信证据转化为长期推荐机制。
- `最小行动`：设计最小补证动作。
